Run with Docker mapping local volumes for development:
`docker run -p 8081:8081 -v /home/lala/code/hot_dog_pwa/:/my-app -it --rm node /bin/bash
`
Then from inside the container:
`cd my-app`
`npm start -- --hostname 0.0.0.0 --port 8081`
